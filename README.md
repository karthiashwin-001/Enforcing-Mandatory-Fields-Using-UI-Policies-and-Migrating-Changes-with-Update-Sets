# Enforcing-Mandatory-Fields-Using-UI-Policies-and-Migrating-Changes-with-Update-Sets
## project overview
In this project, we address this issue by implementing a UI Policy that makes the Priority field mandatory when the State of an incident is set to In Progress. Additionally, we capture and migrate this configuration using Update Sets, enabling seamless deployment across different ServiceNow instances.
